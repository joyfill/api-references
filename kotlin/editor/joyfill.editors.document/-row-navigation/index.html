RowNavigation

data class RowNavigation(val pageId: String, val fieldPositionId: String, val rowId: String, val rowIndex: Int, val scrollIndex: Int, val rowManager: RowManager?, val rowEntry: RowEntry?, val openRowForm: Boolean = false, val animate: Boolean = true) : NavigationTarget

Navigation target for scrolling to a row in a table/collection.

Parameters

pageId

The page containing the field.

fieldPositionId

The field position ID (used to identify the table/collection field).

rowId

The row ID to navigate to (empty string if row doesn't exist).

rowIndex

The pre-computed index of the row within the table (0 if row doesn't exist, -1 for collections).

rowManager

The row manager for the target row, or null if row doesn't exist.

openRowForm

Whether to open the row form modal.

animate

Whether to animate the scroll.

navigationId

Unique identifier for this navigation request.
